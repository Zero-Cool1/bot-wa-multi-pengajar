# 🤖 Bot WhatsApp Multi-Pengajar

Bot WhatsApp open-source berbasis Node.js untuk membantu pengajar menandai santri secara otomatis melalui WhatsApp Web.

## 🎯 Fitur Utama
- **Multi-Pengajar**: Satu server bisa menampung banyak akun bot WhatsApp
- **Tag Otomatis**: Menandai santri di grup WhatsApp dengan format yang rapi
- **Copy-Paste dari Spreadsheet**: Mendukung format TAB dan SPASI
- **Dashboard Web**: Antarmuka mudah digunakan untuk pengajar
- **Keamanan PIN**: Setiap bot dilindungi PIN 6-digit
- **Timeout QR Otomatis**: QR code expired dalam 60 detik jika tidak di-scan

## 🛠️ Tech Stack
- **Runtime**: Node.js 20+
- **Framework Web**: Express.js 4.x
- **WhatsApp Library**: whatsapp-web.js
- **Process Manager**: PM2
- **Tunneling**: Ngrok
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
